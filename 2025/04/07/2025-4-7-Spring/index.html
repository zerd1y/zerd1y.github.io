基本框架：Spring Framework -> 集合：SpringBoot

快速入门

  1. 在 itheima_web_project 下：File -> new Module -> 选择SpringBoot，填写信息
    • Artifact：（Name）springboot-web-quickstart
    • Group：com.itheima
    • Package name：com.itheima
    • Project SDK：JDK 17
    • Java：17 -> Next
  2. Spring Boot 版本：3.4.4
  3. 勾选 Web -> Spring Web -> Create

在 src/main/java/com.itheima. 下新建 controller.HelloController Java文件

运行 SpringbootWebQuickstartApplication，在浏览器输入 localhost:8080/hello 显示页面
