简介

jQuery 是一个轻量级的”写的少，做的多”的 JavaScript 库：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript 特效和动画
  • HTML DOM 遍历和修改
  • AJAX
  • Utilities

安装

2种方法：

  1. 下载jQuery库：jQuery

    在要使用 jQuery 的页面引入.js文件

    路径替换为你的下载路径

  2. CDN导入：

语法

基础语法：

  • 美元符号定义 jQuery
  • 选择符（selector）”查询”和”查找” HTML 元素（和CSS选择器一样）
  • jQuery 的 action() 执行对元素的操作

选择器

  • $("p")：元素选择器
  • $("#test")：id选择器
  • $(".test")：class选择器
  • css可用的，jQuery都可用

事件

事件：页面对不同访问者的响应

事件方法

基本语法：

$(document).ready()

$(document).ready()方法允许我们在文档完全加载完后执行函数

简洁写法:：

click()

click()：当按钮点击事件被触发时会调用一个函数

mouseenter()

mouseenter()：当鼠标指针穿过元素时，会发生 mouseenter 事件

操作 DOM

获取

  • text()：获取（设置）文本内容
  • html()：获取（设置）HTML内容

css()

css() 方法设置或返回被选元素的一个或多个样式属性

  • css("propertynam*","value");设置 CSS 属性

    css({"propertyname":"value","propertyname":"value",...});

脚本：

显示 / 隐藏

  • $(selector).hide(speed,callback);：隐藏
  • $(selector).show(speed,callback);：显示
  • $(selector).toggle(speed,callback);：切换 hide() 和 show() 方法

参数：

  1. speed（可选）：隐藏/显示的速度（”slow”、”fast” 或毫秒）
  2. callback（可选）：完成后所执行的函数名称
