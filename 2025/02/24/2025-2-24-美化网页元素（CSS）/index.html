美化网页元素

感觉前端在最基本的知识框架学完后，需要大量的搬运、模仿、借鉴、总结，再加上一些艺术设计的感觉，才能搭好一个漂亮的网页 -2025/2/26

为什么要美化网页

  1. 有效的传递页面信息
  2. 吸引用户
  3. 凸显页面主题
  4. 提高用户体验

span标签

  • <span>标签：重点突出的字，用审判套起来

字体

字体系列

  • font-family
  • font-family 属性应该设置几个字体名称作为一种”后备”机制，如果浏览器不支持第一种字体，他将尝试下一种字体

字体样式

  • font-style：
    1. normal：正常
    2. italic：斜体

字体大小

  • font-size
  • 大小设置：
    1. px：像素
    2. em：1em=16px

其他

  • font-weight：字体粗细
  • color：颜色

脚本

文本

颜色

  • rgb值：如rgb(255，0，0)

  • rgba值：如rgba(85,63,224,0.5)

    （a值透明度，值在0~1间）

  • 名称：如red

  • 十六进制值：如#FF0000 （0~F）

对齐方式

  • text-align：
    1. centre：居中
    2. right/left：向右/左对齐
    3. justify：左、右外边距对齐（每行宽度相等）

修饰（划线）

  • text-decoration：
    1. none：主要用于删除链接的下划线
    2. overline：上划线
    3. line-through：中划线
    4. underline：下划线（不建议强调指出不是链接的文本，因为这常常混淆用户）

缩进

  • text-indent:
    1. px
    2. em：常用（2em）

阴影

  • text-shadow:
  • 语法：text-shadow: h-shadow v-shadow blur color;
    1. h-shadow：水平阴影（必选）
    2. v-shadow：垂直阴影（必选）
    3. blur：阴影距离（可选）
    4. color：阴影颜色（可选）

其他

  • height：块高

  • line-height：行高

    行高和块高一致，可以让文字上下居中（line-height=height）

  • 文本图片水平对齐：vertical-align:middle

脚本

列表

  • list-style
    1. none：去掉圆点
    2. circle：空心圆
    3. decimal：数字
    4. square：正方形

背景

背景颜色

  • background-color
  • 可以设置渐变色，详见“渐变”一栏

背景图片

  • background-image：

    默认情况下：背景图像进行平铺重复显示，以覆盖整个元素实体

  • background-repeat：

    1. no-repeat：不平铺
    2. repeat-x：水平方向平铺
    3. repeat-y：竖直方向平铺

渐变
