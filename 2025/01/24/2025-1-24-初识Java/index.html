初识Java

语言只是实现目标的工具，而不是目标本身。——Gosling

程序设计语言的成功更多地取决于其支持系统（support system）的能力，而不是语法的精巧性。

Java简介

  • Java 是一种高级的、面向对象的编程语言，以其可移植性、健壮性和可扩展性而闻名。

  • Java 由 Sun Microsystems（现为 Oracle）开发，遵循“一次编写，随处运行”原则，允许代码在任何具有 Java 虚拟机 （JVM） 的设备上运行。

  • 它广泛用于构建大型企业应用程序、Android 移动应用程序和 Web 服务。

  • Java 具有自动内存管理（垃圾回收）、庞大的标准库和强大的安全功能，使其成为后端系统、分布式应用程序和基于云的解决方案的热门选择。

快速入门

编译：javac Main.java

运行：java Main

Java程序的生命周期

  1. 编辑程序

    在文本编辑器中键入程序，以.java扩展名保存

  2. 编译

    javac编译器：输入 Java 源代码，输出字节码（独立于机器或平台），生成.class文件

  3. 执行字节码

    Java虚拟机（JVM）：输入字节码，输出机器码（0和1），由本地机器的 CPU 执行

语法

  • 每一行代码都必须位于class（类）内
  • 类名以大写字母开头

注意：

  1. Java 区分大小写
  2. java 文件的名称必须与类名匹配

输出

System.out.println()

在 main() 方法中，println() 将一行文本打印到屏幕上的方法

System：是一个内置的 Java 类，其中包含有用的成员，例如 out，它是“output”的缩写。println() 方法是 “print line” 的缩写，用于将值打印到屏幕（或文件）

System.out.print()

和上面一样，但不会在输出的末尾插入新行

注释

  • 单行注释：//
  • 多行注释：/*...balabala...*/

变量

类型：String、int、float、char、boolean……

声明：type variableName = value;

命名规则：

  1. 可以包含字母、数字、_和$
  2. 可以$、_、小写字母开头
  3. 区分大小写
  4. 保留字（如 Java 关键字，如 int 或 boolean）不能用作名称

数据类型

基本数据类型：

byte, short, int, long, float, double, boolean,char

非基本数据类型：

String,Array,Class

  • 由程序员创建（String除外）
  • 可用于调用方法
  • 常以大写字母开头
数据类型 描述
byte 2^7~ 2^7 - 1
short -2^15 ~ 2^15 - 1
int -2^31 ~ 2^31 - 1
long -2^63 ~ 2^63 - 1
float 约 7 位小数
double 约 15 位小数
boolean true/false
char 1个字符

注意：

  • long：以L结尾
  • float：以f结尾
  • double：以d结尾

科学计数法：e或E

类型转换

自动转换：

byte -> short -> char -> int -> long -> float -> double

强制转换：必须加()

double -> float -> long -> int -> char -> short -> byte

字符串

String 变量包含一组用双引号括起来的字符

方法：

  • length()：字符串长度

  • toUpperCase()：转换为大写

  • toLowerCase()：转换为小写

  • indexOf("...")：返回字符串中指定文本首次出现的索引（包括空格）

  • concat()：连接两个字符串

  • +：数字相加，字符串串联

转义：

  • \：转换为普通字符串字符
  • \n：换行
  • \t：Tab
  • \r：回车
  • \b：退格
  • \f：换页符

Math

Math.max(x,y)

Math.min(x,y)

Math.sqrt(x)

Math.abs(x)

Math.random()：返回一个介于 0.0（含）和 1.0（不含）之间的随机数

条件判断

  • if…else
  • 简写 if
  • switch

循环

  • while
  • do…while
  • for
  • for…each：专门用于遍历数组（或其他数据集）中的元素

数组

  1. 创建：用方括号定义变量类型

  2. .length：数组长度

  3. 遍历：

    • for 循环
    • for-each 循环

多维数组

  1. 创建：

  2. 遍历：

    • for 循环
    • for-each 循环

方法

方法是一个代码块，仅在调用时运行，您可以将数据（称为参数）传递到方法中。

方法用于执行某些作，它们也称为函数。

创建

  • myMethod()：方法的名称
  • static：表示该方法属于 Main 类，而不是 Main 类的对象
  • void：表示该方法没有返回值

调用

Method();

重载

重载：多个方法可以具有相同的名称但具有不同的参数

  1. 方法名相同。

  2. 参数列表不同：

    • 个数不同
    • 类型不同

作用域

方法作用域（局部变量，Method Scope）

  • 变量在方法内部声明，只能在该方法内访问

块作用域（Block Scope）

  • 在代码块 {} 内部定义的变量，作用范围仅限于该代码块

全局作用域（类变量 / 静态变量）

  • 由 static 关键字修饰，属于整个类，而不是某个对象
  • 在类加载时初始化，并在整个程序运行期间存在
  • 任何方法都可以访问它（如果是 public）

实例作用域（实例变量）

  • 属于某个对象，每个对象都有自己独立的实例变量。

  • 只有在对象被创建后，实例变量才会存在。

  • 只能通过对象访问实例变量。

循环作用域（Loop Scope）

  • 在 for、while、do-while 循环内部定义的变量，作用范围仅限该循环内
