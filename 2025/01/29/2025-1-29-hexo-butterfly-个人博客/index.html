小tips

  1. 本地编辑完，本地部署3连：

    • hexo clean
    • hexo generate
    • hexo server

  2. 部署到vercel上自己的服务器：

    先在个人blog文件夹用git bash打开

    • hexo clean
    • hexo generate
    • hexo deploy

  3. background图片部署：

    • 用网页上的图片（不要用本地的相对地址，绝对地址）
    • 直接在background后复制链接，不要在链接外面加url()

  4. hexo deploy部署过程停滞不前

    • 可能是网络原因
