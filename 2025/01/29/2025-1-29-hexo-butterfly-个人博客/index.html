小tips

  1. 本地编辑完，本地部署3连：

    • hexo clean
    • hexo generate
    • hexo server

  2. 部署到vercel上自己的服务器：

    先在个人blog文件夹用git bash打开

    • hexo clean
    • hexo generate
    • hexo deploy

  3. background图片部署：

    • 用网页上的图片（不要用本地的相对地址，绝对地址）
    • 直接在background后复制链接，不要在链接外面加url()

图片显示

此处讨论仅仅在Typora中创建博文，图片导入的形式

  1. 纯Markdown - 相对路径：

    • 自己手打，我的路径一般 ../img/screenshot/... （source 文件下的相对路径）
    • 如果你的博文名含中文，只能用这种方法

  2. 纯HTML：

    • 设置缩放，不需要的话改成 100%

    • 这种方式，博文名必须是非中文，因为生成 public 渲染文件时中文路径会转换成乱码导致无法渲染
